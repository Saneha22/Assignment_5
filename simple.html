Different ways to add CSS:

  • Inline CSS :
    In inline CSS, the styles add with html element. We add the styles directly at the time of declaration of html element.
    Inline CSS has more priority than internal.

  • Internal CSS:
    In internal CSS, the styles are added to the 'style' tag in head element
    Internal CSS has the highest priority.

  • External CSS:
    In external CSS, the styles are added in different file which connect to the html file using the 'link' tag
    External CSS has the lowest priority.
